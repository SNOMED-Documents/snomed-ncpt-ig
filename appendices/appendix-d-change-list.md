# Appendix D: Change List

### Purpose

<br>

This appendix documents all SNOMED CT concept additions, remaps, semantic tag updates, international extension substitutions, and omissions that differ from the 2020 NCPT baseline release.

The appendix reflects changes introduced in:

* April 2025 NCPT Refset release
* April 2026 NCPT Refset release

The purpose is to support:

* implementation migration from legacy 2020 NCPT mappings
* terminology governance review
* analytics and reporting refset maintenance
* EHR build validation
* interoperability regression testing
* auditability of NCPT concept evolution
