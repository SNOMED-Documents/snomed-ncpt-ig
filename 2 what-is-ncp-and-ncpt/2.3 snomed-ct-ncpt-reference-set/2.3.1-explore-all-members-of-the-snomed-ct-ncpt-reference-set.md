# 2.3.1 Explore all Members of the SNOMED CT NCPT Reference Set

# Demonstration Video

The following video shows how to view the members of the NCPT reference set using the SNOMED International SNOMED CT Browser. You can access the browser at [browser.snomedtools.org](http://browser.snomedtools.org).

  

# Step-by-step Instructions

  

This step-by-step guide will show you how to view the members of the NCPT reference set using the SNOMED International SNOMED CT Browser.

### 1\. Open the SNOMED CT Browser ([browser.snomedtools.org](http://browser.snomedtools.org))

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2FrBhQLMcn3tQUh51QfFpo6X_doc.png?alt=media&token=95abde68-5a32-4b14-907e-63f9102c69f0" alt="Open the SNOMED CT Browser (browser.snomedtools.org)" title=""></figure>

### 2\. Click "Go browsing...International Derivatives"

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2Fj13HkEtprJWiH1kxUwA45M_doc.png?alt=media&token=586d5019-b68c-4f1b-b46c-61d33d17e7ae" alt="Click 'Go browsing...International Derivatives'" title=""></figure>

### 3\. Click "Refset"

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2F7Su6oCGefc6soAZN5sRRht_doc.png?alt=media&token=5eab3b23-9793-47c6-9b25-6a43388329f7" alt="Click 'Refset'" title=""></figure>

### 4\. Scroll down the list to see the "Nutrition Care Process Terminology reference set" - then click this row

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2Fkg9G5UUL6zBvDKUy6seZdL_doc.png?alt=media&token=6ce788dd-f81e-4622-84f1-d1f0b98ca8a2" alt="Scroll down the list to see the 'Nutrition Care Process Terminology reference set' - then click this row" title=""></figure>

### 5\. Click "Members"

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2FmEgVPjzpJnTKv1aU2B1zf8_doc.png?alt=media&token=e7efed5c-bf51-4adc-bce1-5fb598851fd8" alt="Click 'Members'" title=""></figure>
