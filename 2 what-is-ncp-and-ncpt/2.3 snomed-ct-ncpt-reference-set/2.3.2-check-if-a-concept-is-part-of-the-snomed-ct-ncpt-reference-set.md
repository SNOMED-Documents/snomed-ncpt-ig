# Check if a Concept is Part of the SNOMED CT NCPT Reference Set

## Demonstration Video

The following video shows how to verify if a given concept is a member of the NCPT reference set using the SNOMED International SNOMED CT Browser. You can access the browser at [browser.snomedtools.org](http://browser.snomedtools.org/).

## Step-by-Step Instructions

Explore how to verify if a concept is part of the NCPT reference set using the SNOMED International SNOMED CT Browser. You can access the browser at [browser.snomedtools.org](http://browser.snomedtools.org/).

#### 1. Open the SNOMED CT Browser ([browser.snomedtools.org](http://browser.snomedtools.org))

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2FrBhQLMcn3tQUh51QfFpo6X_doc.png?alt=media&#x26;token=95abde68-5a32-4b14-907e-63f9102c69f0" alt="Open the SNOMED CT Browser (browser.snomedtools.org)"><figcaption></figcaption></figure>

#### 2. Click "Go browsing...International Derivatives"

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2F5S5xM3xqrnqgZuganpSj2E%2Fj13HkEtprJWiH1kxUwA45M_doc.png?alt=media&#x26;token=586d5019-b68c-4f1b-b46c-61d33d17e7ae" alt="Click &#x27;Go browsing...International Derivatives&#x27;"><figcaption></figcaption></figure>

#### 3. Perform a search for a concept of interest.

Fill in a search term matching the concept you are looking for.

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2FebQs7m9rhYMrdTvwGWpkvn_doc.png?alt=media&#x26;token=755056a5-96f7-4942-8691-d9ac19f225ed" alt="Perform a search for a concept of interest."><figcaption></figcaption></figure>

#### 4. Select the concept of interest

Click on the concept, so that its details appear in the Summary section

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2FgXUff7cdXZgdUPjjR5sDs3_doc.png?alt=media&#x26;token=03277f24-856c-4667-a912-10846ba3d2a0" alt="Select the concept of interest"><figcaption></figcaption></figure>

#### 5. Click "Refsets"

Select the "Refsets" tab

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2Fn9fNRXpjD3pxi6aNuPJP3k_doc.png?alt=media&#x26;token=ab04c5b3-7801-4ba5-82d8-e670ed68b7cc" alt="Click &#x27;Refsets&#x27;"><figcaption></figcaption></figure>

#### 6. View simple type reference sets

If you see the concept |Nutrition Care Process Terminology reference set| in the list of Simple Refsets Memberships, this means that the selected concept is included in the reference set.

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2FweZz1fN2ocjscJ1o5Q31DN_doc.png?alt=media&#x26;token=e3969279-4ff7-4b3d-a36d-bf31ed9cce73" alt="View simple type reference sets"><figcaption></figcaption></figure>

#### 7. Here you see what simple type reference sets, the selected concept is included in

As you can see here, when selecting the concept 414916001 |Obesity (disorder)|, this is part of the reference set.

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2Fw3xnphEimW1z3HviBpeELd_doc.png?alt=media&#x26;token=033941a1-3962-4938-843f-9d694dcbcc78" alt="Here you see what simple type reference sets, the selected concept is included in"><figcaption></figcaption></figure>

#### 8. In this example, the concept 'Hypoglycaemic coma' is selected

Try to search for and select the concept [267384006 |Coma due to hypoglycemia (disorder)|](http://snomed.info/id/267384006)

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2FddE6EATSyTb42cMkMD4Doi_doc.png?alt=media&#x26;token=33b11719-2b78-4634-b551-aaa1b36f8f08" alt="In this example, the concept &#x27;Hypoglycaemic coma&#x27; is selected"><figcaption></figcaption></figure>

#### 9. Click "Refsets"

Click on "Refsets"

<figure><img src="https://static.guidde.com/v0/qg%2FiKQu6ToHloOCD99Y6AVoQVimxPj1%2FenFmVat3xacwPQ2ik46y4L%2Fc5TxKzXCr7CujTdNNAyiUk_doc.png?alt=media&#x26;token=66e8079c-4d63-437c-8f7f-7ed84e4605bf" alt="Click &#x27;Refsets&#x27;"><figcaption></figcaption></figure>

#### 10. This concept is NOT included in the NCPT reference set

As you can see, the |Nutrition Care Process Terminology reference set| does not appear in the list of Simple Refsets Memberships, indicating that the selected concept is NOT part of the reference set.
